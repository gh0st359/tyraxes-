# Privilege escalation

Skill: `/privilege-escalation`

## Goals

- Enumerate local privilege boundaries
- Identify misconfigurations / creds / services / kernel/userland paths
- Prove impact with minimal persistence
- Save proofs under artifacts

- start privilege

- cleanup notes
