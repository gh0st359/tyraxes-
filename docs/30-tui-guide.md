# TUI guide

Tyraxes keeps the full-screen terminal UX:

- scrollback + streaming responses
- tool approval / always-approve modes

## Offensive operator tips

- Pin the red-team agent before deep work
- Keep a second pane/shell for manual validation if desired
- Use todos/plan tools for kill-chain status
- Periodically ask the agent to refresh `attack-graph.md`
